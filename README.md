# 🏏 IPL Data Analysis — Matplotlib & Seaborn

A data visualization project exploring IPL match trends, 
team performance, and player statistics.

## 📌 About
This project explores a synthetically generated IPL-style 
dataset (500 matches, 2008–2023) through 5 key visualizations.

## 📊 Visualizations Included
1. 🏆 Team Win Count — Horizontal Bar Chart
2. 🏏 Toss Decision Analysis — Pie Chart
3. 🌟 Top Scorers — Bar Chart
4. 📅 Matches Per Season — Line Chart
5. 🔥 Venue vs Season — Heatmap

## 🛠️ Libraries Used
- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn

## ▶️ How to Run
Open the `.ipynb` file in Google Colab (Recommended) or Jupyter Notebook
